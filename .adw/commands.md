# ADW Project Commands

## Package Manager
npm

## Install Dependencies
npm install

## Run Linter
npm run lint

## Type Check
npx tsc --noEmit

## Additional Type Checks
npx tsc --noEmit -p adws/tsconfig.json

## Run Tests
npm test

## Run Build
npm run build

## Start Dev Server
npm run dev

## Prepare App
npm install && npx next dev --port {PORT}

## Run E2E Tests
npx playwright test

## Library Install Command
npm install

## Script Execution
npx tsx
